# Trigram frequency lists and validation scripts

The trigram frequency lists were created with opt (Andreas Wettstein, AdNW) with corpora from the University of Leipzig.

To run the evaluation put the .3 and .txt files in the same folder. The script will search for each language.3 - language.txt pair automatically.

An additional script calculates partial (sub) wordlists and another calculates the percentage compared to the top 200 words in each language.
